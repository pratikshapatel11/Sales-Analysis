Power BI Dashboard analyzing sales performance, product trends, and regional sales insights using interactive visualizations.

📊 Sales Analysis Dashboard – Power BI

📌 Project Overview

This project focuses on analyzing sales data and building an interactive dashboard using Microsoft Power BI.

The dashboard helps businesses understand their sales performance through clear visualizations and insights.

The main goal of this project is to convert raw sales data into meaningful insights that support better business decisions.

🎯 Objectives

Analyze overall sales performance

Identify top-performing products

Track monthly sales trends

Compare sales across regions

Understand profit and revenue patterns

🛠 Tools & Technologies
Tool	Purpose
Microsoft Power BI	Data Visualization
Microsoft Excel	Data Source
Power Query	Data Cleaning
DAX	Data Calculations
📂 Dataset

The dataset includes the following fields:

Order ID

Order Date

Customer Name

Product Category

Region

Sales Amount

Quantity

Profit

Data was cleaned and transformed before visualization.

📊 Dashboard Features

✔ Total Sales KPI
✔ Total Profit Analysis
✔ Sales by Region
✔ Top Selling Products
✔ Category-wise Sales
✔ Monthly Sales Trend

🔍 Key Insights

Some regions generate higher revenue than others

Certain product categories contribute majority of sales

Sales follow monthly seasonal trends

Profit varies depending on product category
